# Mithril Starter Kit
A boilerplate Mithril application using ES6, Babel, Webpack 4, Sass/SCSS, Webpack dev server hot reload and eslint

# Getting Started
Install Yarn
```
$ sudo apt-get update
$ sudo apt-get install yarn
```

Build the application
```
$ yarn 
```

Run the application
```
$ yarn start
```

In your browser, go to 
`http://localhost:3000/`
